Authentication Gateway

System Notice

This system provides Discord role verification for UPL community members. Authentication is performed via Google OAuth 2.0 protocol.

Access Instructions

  1. Execute /verify command in Discord
  2. Authenticate using your Google account
  3. Receive automatic role assignment upon approval

Approval Process

  • @wisc.edu addresses: Automatic approval
  • Other domains: Manual review (24-48 hours)
  • Role assignment: Automatic upon approval
